🔥 Tentang Aplikasi Ini

Fire Detection adalah aplikasi deteksi kebakaran real-time yang menggunakan model kecerdasan buatan YOLOv9 (You Only Look Once v9). Aplikasi ini dirancang untuk mendeteksi keberadaan api dalam gambar dengan akurasi tinggi.

Teknologi ini sangat berguna untuk sistem pemantauan kebakaran otomatis, deteksi dini, dan analisis citra satelit untuk pencegahan bencana.
